# 🎬 OTT Website

A modern, responsive **OTT (Over-The-Top) Streaming Platform** built using **HTML5, CSS3, and JavaScript (ES6)**. This project is inspired by popular streaming platforms like **Netflix**, **Amazon Prime Video**, and **Disney+**, providing users with an engaging interface to browse movies, TV shows, categories, search content, and manage their personal watchlist.

---

## 📖 Project Overview

The OTT Website is a frontend web application that simulates a real-world streaming platform. Since this project does not use a backend, movie and TV show information is managed using static JSON/JavaScript objects, while browser Local Storage is used to store user-specific data such as wishlists and recently viewed content.

The application is fully responsive and optimized for desktop, tablet, and mobile devices.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Saritha-kamatham/OTT-Website.git
```

### Navigate to the Project

```bash
cd OTT-Website
```

### Run the Project

Simply open **index.html** in your preferred web browser.

No installation or additional dependencies are required.
