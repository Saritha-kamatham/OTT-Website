About CineStream

Welcome to CineStream, a premier frontend-focused OTT web application showcase built with a high-performance Spring Boot backend.

Immersive Design

Features vibrant glassmorphic designs, responsive typography, micro-interactions, and dark/light layouts designed to mimic modern streaming giants.

Robust Architecture

Driven by a Spring Boot Rest API providing dynamic search, filtering, and detail lookups of movies and series instantly.

Local Persistence

Leverages Local Storage to store your personal watchlist ("My List"), track playing progress ("Continue Watching"), and store viewing history.

Project Tech Stack

  • Backend Core: Java 21, Spring Boot 3.3.x (Web, DevTools)
  • Template Engine: Thymeleaf (HTML5 integration, fragments caching)
  • Styling: CSS3 (Custom properties, grid, flexbox, keyframes, transitions)
  • Frontend Logic: JavaScript ES6 (Asynchronous fetch, Local Storage API, event emission)
  • Vector Assets: FontAwesome CDN v6.4.0
